Include method

Using .indexOf(item,start) as a condition
arr.indexOf(element) === -1

This method is similar to the include method in Ruby.
If item is not in the array, -1 is returned.
Can be used as a conditional to test if element is inside of the array.

Accepts two parameters: item and start

Can be used on an Array or String

Item: The element you're searching for. This is required*

Start: Where to start the search. Negative values will start at the given position counting from the end, and search to the end. *Optional argument

Example:

var arr = [1,2,3,4,5,6];

if (arr.indexOf(3) !== -1){
console.log("It is inside of the array") }
else{ console.log("It's not!")
}
// => returns "It is inside of the array"

Building a function for an Include method
function include(item,dataStructure){
return (item in dataStructure) ? true : false;
}

Example:

var arr = [1,2,3,4,5,6];
include(5,arr) // => true
